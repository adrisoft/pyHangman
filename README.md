# pyHangman
### The famous game Hangman in Python.

Requiriments:
* Python 3.x is required

Prompts to install:
1. python3 -m venv .env
2. source .env/bin/activate
3. pip install -r requirements.txt
4. chmod u+x pyHangman.py
5. ./pyHangman.py
6. Enjoy!
